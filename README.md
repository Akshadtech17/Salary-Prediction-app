## 🧠 About

> A **machine learning-powered web application** that predicts an individual's salary based on key professional attributes. Built as a complete end-to-end ML system — from raw data ingestion to a live REST API.

This project demonstrates real-world ML engineering: preprocessing pipelines, model comparison, hyperparameter tuning, explainability via SHAP, and production-grade API deployment.

---

## ✨ Key Features

| Feature | Description |
|--------|-------------|
| 🎯 **Smart Predictions** | Estimates salary from experience, education, role, location & industry |
| 🔬 **Full ML Pipeline** | CSV → Feature Engineering → Train → Evaluate → Serialize → Serve |
| 📊 **Explainability** | SHAP values reveal which factors drive each prediction |
| ⚡ **REST API** | FastAPI/Flask with Swagger docs, validation & Docker support |
| 🧪 **Model Comparison** | Benchmarks 4+ algorithms; XGBoost wins with 93.2% R² |
| 🐳 **Docker Ready** | One-command containerized deployment |
